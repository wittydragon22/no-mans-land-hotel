# Logo Image

Please place your logo image file here as `logo.jpg`.

The logo should be:
- Format: PNG (with transparency preferred)
- Size: At least 40x40 pixels (will be scaled to 40x40 in the app)
- Recommended: 80x80 or 120x120 pixels for better quality on high-DPI displays

The logo component will automatically use this image if it exists, otherwise it will fall back to an SVG version.
