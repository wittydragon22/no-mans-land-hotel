# Room Images

请按房型组织图片，每个房型共享相同的图片集。

## 目录结构

```
public/images/rooms/
├── standard/     # Standard 房间图片（001-004 共享）
│   ├── 1.jpg
│   ├── 2.jpg
│   ├── 3.jpg
│   └── 4.jpg
├── deluxe/       # Deluxe 房间图片（005-007 共享）
│   ├── 1.jpg
│   ├── 2.jpg
│   ├── 3.jpg
│   └── 4.jpg
└── suite/        # Suite 房间图片（008-010 共享）
    ├── 1.jpg
    ├── 2.jpg
    ├── 3.jpg
    └── 4.jpg
```

## 图片命名规则

### 按房型组织

每个房型有自己的文件夹，所有该房型的房间共享相同的图片：

- **Standard 房间** (001-004) → `standard/` 文件夹
  - `1.jpg`, `2.jpg`, `3.jpg`, `4.jpg`
  
- **Deluxe 房间** (005-007) → `deluxe/` 文件夹
  - `1.jpg`, `2.jpg`, `3.jpg`, `4.jpg`
  
- **Suite 房间** (008-010) → `suite/` 文件夹
  - `1.jpg`, `2.jpg`, `3.jpg`, `4.jpg`

### 图片数量

- 每个房型建议 **4 张图片**
- 图片会自动轮播显示（每 5 秒切换）
- 鼠标悬停时显示导航箭头和指示器
- 可以点击指示器或箭头手动切换

## 图片要求

- **格式**: JPG, PNG, 或 WebP（推荐 JPG）
- **推荐尺寸**: 1200x800 像素（16:10 宽高比）
- **最小尺寸**: 800x600 像素
- **文件大小**: 建议每个文件小于 500KB（可以使用图片压缩工具优化）
- **命名**: 使用数字命名：`1.jpg`, `2.jpg`, `3.jpg`, `4.jpg`

## 功能特性

- ✅ 自动轮播（每 5 秒切换）
- ✅ 鼠标悬停显示导航控件
- ✅ 点击箭头切换图片
- ✅ 点击指示器跳转到指定图片
- ✅ 如果图片不存在，自动显示占位符

## 示例

如果你有 4 张 Standard 房间的图片：
1. 创建 `public/images/rooms/standard/` 文件夹
2. 将图片命名为 `1.jpg`, `2.jpg`, `3.jpg`, `4.jpg`
3. 所有 Standard 房间（001-004）都会显示这 4 张图片的轮播

同样地，Deluxe 和 Suite 房间也按照相同的方式组织。
